# TPSAS R package 

R Package for twopiece sinh arcsinh distributions

To install this package, use

```
library(devtools)
install_github("FJRubio67/TPSAS")
```

For more information, see the following illustrative example:

1. [TPSAS Package
](https://rpubs.com/FJRubio/TPSAS)
